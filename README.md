## 使用

### 使用命令行
```bash
$ cd ant_snaphoenix_fe
$ npm install
$ npm start         # 访问 http://localhost:8000
```

### 使用 docker

```bash
// dev 
$ npm run docker:dev

// build 
$ npm run docker:build


// production dev 
$ npm run docker-prod:dev

// production build 
$ npm run docker-prod:build
```
